Portfolio is a simple and modern website template for developers. You can use it to show your skills, projects, and experience in an attractive way.  
It's made using React, Vite, Tailwind CSS, and Firebase, so it looks nice and works quickly on any device.

## Getting Started

### Prerequisites

Make sure you have these before starting:

- **Programming Language:** JavaScript
- **Package Manager:** npm

### Installation

Follow these steps to get your portfolio up and running:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShamJohari20/Portfolio
   ```

2. **Go to the project folder:**
   ```bash
   cd Portfolio
   ```

3. **Install all the required packages:**
   ```bash
   npm install
   ```

---

### Usage

To start your portfolio website, run this command:

**Using [npm](https://www.npmjs.com/):**
```bash
npm run dev
```
